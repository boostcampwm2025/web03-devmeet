# vpc

## vpc 이름

### clobby-vpc
- 설명: clobby 서비스에 전반적인 부분을 담당하고 있다.

## subnet 종류

### clobby-service-subnet
- 사용하는 서버들이 모여 있는 서브넷

### clobby-infra-subnet
- 설명: infra들이 모여있는 서브넷

## server 종류

### clobby-infra-server-1
- 설명: clobby에서 사용되는 mysql, redis, prometheus, grafana등이 사용되는 서버

### clobby-test-server
- 설명: clobby에 테스트 서버가 위치하는 서버

### clobby-deploy-server
- 설명: clobby에 배포 서버가 위치하는 서버
